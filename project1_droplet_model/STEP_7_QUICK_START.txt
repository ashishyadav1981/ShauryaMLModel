## If Something Goes Wrong

### Error: "best_model.pth not found"
```powershell
# Step 6 didn't complete. Run it first:
python3 step6_training_loop.py
```

### Error: "ModuleNotFoundError"
```powershell
# Try:
pip install torch matplotlib scikit-learn
python3 step7_evaluate.py
```

### Error: "FileNotFoundError: X_test.npy"
```powershell
# Run data split first:
python3 step4_data_split.py
python3 step7_evaluate.py
```

### Script runs but produces weird results
```
Check: evaluation_plots.png
  - If all points scattered randomly: model didn't learn
  - Return to Step 6, check training_history.png
  - If training curve shows overfitting, re-train with more dropout
```

## Key Metrics Quick Reference

After running, you'll see metrics like:

| Metric | Meaning | Good Value |
|--------|---------|------------|
| R² | % of variance explained | > 0.85 |
| MAE | Average error (physical units) | < 10% of range |
| RMSE | Error with outliers emphasized | Similar to MAE |
| MSE (norm) | Squared error in 0-1 space | < 0.005 |
